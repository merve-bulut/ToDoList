# ✅ To-Do List App

This is a **simple to-do list application** built using **SwiftUI**.  
With this app, you can:  
✅ Add new tasks.  
✅ List all tasks.  
✅ Delete tasks from the list.  

## 🚀 How to Run the Project?
1. Open the project in **Xcode 16.2+**.  
2. Run the project (**⌘ + R**).  
3. Start adding and managing your tasks!
